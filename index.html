Index es el Nombre predeterminado para la página principal del sitio web. Esta es el cuerpo de la pagina web.\n doctype especifica que el sitio esta construido sobre HTML 5

A continuación se describen etiquetas importantes para la construcción de una página WEB

  • Lang: determina el idioma empleado en cualquier paguna web, señala el lenguaje del documento principal.
  • Head: contiene identificadores de la pagina web.
  • body: es donde se concentra todo lo que el usuario puede interactuar.
  • las etiquetas de cierre siempre comienzan con el simbolo / al inicio de estas.
  • Si bien las etiquetas no son sensibles a los mayúsculas y minúsculas, si lo son los identificadores que descrben a tales etiquetas.
  • span: Agrega un estilo a la etiqueta pero requiere definir dicho atributo dentro de si.
  • ul y ol: Generan una lista con cuyos elementos se describen con li.
  • a: ancor, agrega un enlace que puede ser interno o externo
